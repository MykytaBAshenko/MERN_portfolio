### Installation:
>**1. npm install **
>**2. npm start (before start create your mongodb link(mongodb+srv://name:pass@cluster.mongodb.net/<dbname>?retryWrites=true&w=majority))**
>**3. open new Terminal**
>**4. cd client && npm install**
>**5. npm start**
